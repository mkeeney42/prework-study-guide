HTML

  • The head element contains information about the webpage.
  • The title element is not considered to be in the flow of text but is shown.
  • The body element represents the visible content shown to the user.

CSS

  • When a developer wants to style an element directly in the HTML file, they use inline CSS
  • This internal style sheet contains CSS rules for the webpage in the head section of the HTML file
  • the most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file
